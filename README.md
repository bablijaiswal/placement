# UI Replication Project

This project is a frontend UI replication task created using HTML, CSS, and JavaScript.
The goal of this project was to recreate the provided Figma design as closely as possible by following the layout, spacing, typography, colors, and overall visual hierarchy.

---

## Features

* Responsive Hero Section
* Large Banner Layout
* Dark Blue Content Section
* Product Card Grid
* Reusable Product Cards using JavaScript
* Slider Pagination Dots
* Clean Layout Structure
* Flexbox and CSS Grid Usage

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
